# Dash
This is the local companion plugin for the additional features (addons) in Dash Plus, Dash Pro and Dash Premium.

## Addons
This sections provides an overview about the addons that are included for the paid tiers.

### Dash Plus
- categories
- content
- courses
- dashboard

### Dash Pro
_Includes Plus addons_
- activities
- badges
- calendar_events
- certificates
- course_completions
- course_enrols
- course_sections
- logstore
- myprofile
- programs
- skill_graph

### Dash Premium
_Includes Pro addons_
- activity_completion
- developer
- learningpath
- roleassignments

## Addons from other plugins
Additional data sources and widgets can be implemented by other plugins. These will automatically be available if the Dash Block is installed. Datas ources are restricted though to the table layout if the local plugin is not installed.

### Video Time Pro
- Sessions
- Statistics

### Magic Pro
- Campaigns

### Skills Premium
- Level visuals
- Progress
